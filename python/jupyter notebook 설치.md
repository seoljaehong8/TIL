# jupyter notebook

### 단축키

- `b`(below)
  -  셀을 아래에 한 줄 추가
- `a`(above)
  - 셀을 위에 한 줄 추가 
- x
  - 셀 삭제

- `ctrl + enter`
  - 실행 단축키

- `m`(markdown)
  - 셀을 마크다운 문법 작성 가능
- y
  - 셀을 코드 문법으로 작성 가능
- `h`(help)
  - 단축키에 대한 설명
